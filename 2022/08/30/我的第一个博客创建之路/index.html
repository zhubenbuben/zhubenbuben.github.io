我的第一个博客创建之路

第一章 准备工作
官网下载Node.js www.nodejs.org 附带安装npm管理器
node -v 查看node当前版本
npm -v 查看当前npm版本
安装淘宝镜像源:
npm install -g cnpm –registry=https://registry.npm.taobao.org
安装hexo:
cnpm install -g hexo-cli
hexo -v 查看当前hexo版本

第二章 创建本地blog
选择一个盘并创建blog文件夹:
e:
mkdir blog
进入blog文件夹:
cd blog/
初始化一个blog:
hexo init
查询生成的文件目录:
dir
启动hexo:
hexo s
创建blog:
hexo n “我的博客”
使用编辑器对source_posts文件夹的.md文件进行编辑即是编辑个人博客
清理当前进程
hexo clean:
重新生成blog:
hexo g
重新启动blog:
hexo s

第三章 网页端推送
在GitHub上创建仓库并以本人账号命名 例如 someone.github.io
在cmd中安装git插件:
cnpm install –save hexo-deployer-git
将blog与GitHub链接
打开blog目录下的_config.yml文件 在deployment中进行修改:
type: git
repo: github仓库链接（https与ssh皆可）
branch: main
部署到远端:
hexo d

第四章 自定义界面
在blog目录下克隆想要的主题模式
在_config.yml中修改them为导入的主题名称
hexo clean
hexo g
hexo s
hexo d
